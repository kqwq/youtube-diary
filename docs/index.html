About Diary Explorer

This is a simple web app that allows you to explore the YouTube comments of this video. You can search for an author and then see the videos of that author. You can also see the diary-related comments of the author organized by day.

This app is built with the help of the YouTube Data API v3 and Bootstrap v5.0.2.

This app is open source and you can find the source code on GitHub.
